Imweb SDK Test — Guest Access Block

URL 쿼리 파라미터로 제어: ?mode=guestBlocked|guest|member, ?partnerId=... (테스트 스토어 partnerId). 로그인 URL은 백엔드 experimentalData[memberOnlyAccess].value 그대로 통과. 비어있으면 SDK가 현재 페이지 URL로 fallback 처리.

검증 포인트:

  • Guest + Block ON: 플로팅 버튼 클릭 시 채팅 자리에 "로그인 후 사용 가능합니다" 화면 + "로그인 하기" 버튼이 표시
  • Guest + Block OFF: 일반 채팅 iframe이 로드 (chat web 응답에 따라 달라질 수 있음)
  • Member: 일반 채팅이 정상 노출, 블락 화면 없음
